## 知見・失敗事例メモ
- `pnpm test` 実行時に corepack が `pnpm@9.15.0` を取得しようとしてネットワーク制限で失敗する場合がある（403）。必要に応じて事前に依存関係を用意するか、実行環境のネットワーク許可を確認する。
- Windows PowerShell では環境によって `cmd1 && cmd2` のような `&&` 連結が構文エラーになることがある。連続実行は `;` と `$LASTEXITCODE` で制御する。
