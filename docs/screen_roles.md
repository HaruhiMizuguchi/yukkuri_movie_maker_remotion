# 画面と役割一覧

本ドキュメントは、Web GUI の画面構成と役割を整理したものです。

## 画面一覧（8画面）

1. **ダッシュボード / プロジェクト一覧**
   - 直近ジョブのステータス一覧
   - 新規プロジェクト作成への導線
   - 失敗ジョブのアラート/再実行導線

2. **プロジェクト作成ウィザード**
   - テーマ入力（任意）
   - 生成モード選択（full / scriptOnly / renderOnly）
   - 主要設定の簡易入力（モデル、音声プリセット、解像度など）

3. **プロジェクト詳細 / ワークフロー進捗**
   - 13ステップの進捗一覧と状態表示
   - 各ステップのログ/成果物概要
   - 失敗ステップの再実行ボタン

4. **台本編集**
   - 生成済み台本の編集
   - 話者・感情・長さの調整
   - 台本のバージョン管理（将来拡張）

5. **素材管理**
   - 立ち絵・背景・BGM・SE のアップロード/差し替え
   - キャラ設定（声・立ち絵・表情）
   - 素材のタグ付け/カテゴリ分け

6. **タイムライン編集（動画編集ビュー）**
   - Premiere/DaVinci 的なタイムライン表示
   - 音声・字幕・立ち絵・背景・BGM・SE のレーン表示
   - クリップの並べ替え・開始位置調整・長さ調整
   - 主要パラメータの編集（フェード、トランジション、字幕スタイル）
   - プレビュー再生（区間再生/ループ）

7. **プレビュー & レンダリング**
   - Remotionプレビュー（区間/低負荷）
   - 本番レンダリング実行
   - 出力ファイルのダウンロード

8. **設定**
   - APIキー設定（Gemini/OpenAI など）
   - TTSサーバー設定
   - 出力解像度/品質プリセット
   - 保存先（outputs）の運用設定

## 画面間の主な導線

- ダッシュボード → プロジェクト作成ウィザード
- ダッシュボード → プロジェクト詳細
- プロジェクト詳細 → 台本編集 / 素材管理 / タイムライン編集 / プレビュー & レンダリング
- タイムライン編集 → プレビュー & レンダリング（プレビュー出力）
- 設定 → 各画面に反映

## 役割まとめ

- **生成系（AI/ワークフロー）**: 作成ウィザード、進捗詳細
- **編集系（人手修正）**: 台本編集、素材管理、タイムライン編集
- **確認/出力系**: プレビュー & レンダリング
- **基盤設定系**: 設定画面
