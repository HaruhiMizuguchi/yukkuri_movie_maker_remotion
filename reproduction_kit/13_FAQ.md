# 13. FAQ（よくある質問）

システムの使用に関してよくある質問と回答をまとめています。

## 一般的な質問

### Q1: このシステムは完全無料で使えますか？

**A**: システム自体は無料ですが、外部APIの利用には費用がかかる場合があります。
- **Google Gemini**: 無料枠があり、開発段階では無料で使用可能
- **AIVIS Speech**: **無料（ローカルサーバーとして動作）**
- 詳細は [11_Cost_Estimation.md](11_Cost_Estimation.md) を参照してください。

### Q2: プログラミングの知識がなくても使えますか？

**A**: 基本的な使用には最小限のコマンドライン操作のみが必要です。
ただし、カスタマイズや拡張には TypeScript の知識があると便利です。

### Q3: Windows/Mac/Linux のどれで動きますか？

**A**: すべてのOSで動作します。
- **Windows**: 10/11
- **macOS**: 最新版推奨
- **Linux**: Ubuntu 20.04以降推奨

### Q4: 生成された動画の著作権はどうなりますか？

**A**: 生成された動画の著作権は基本的にあなたに帰属しますが、以下の点に注意してください：
- 使用したAIサービスの利用規約を確認してください
- 東方Projectなどの二次創作の場合は、元作品のガイドラインを遵守してください
- BGMや効果音の著作権にも注意してください

## インストール・セットアップ

### Q5: FFmpegのインストールがうまくいきません

**A**: 
- **Windows**: 環境変数 `Path` にFFmpegの `bin` フォルダを追加後、**コマンドプロンプトを再起動**してください
- **Mac**: `brew install ffmpeg` を実行してください
- **Linux**: `sudo apt install ffmpeg` を実行してください

詳細は [06_Troubleshooting.md](06_Troubleshooting.md) を参照してください。

### Q6: Node.jsのバージョンはどれを使えばいいですか？

**A**: TypeScript（Node.js）版を前提とします。Node.jsは **LTS（推奨: 20以上）** を使用してください。
```bash
node --version
```
で確認できます。

### Q7: 依存関係のインストールでエラーが出ます

**A**: 以下を試してください：
```bash
# pnpmで依存関係を再インストール
pnpm install
```

それでも解決しない場合は、`node_modules` を削除してから再インストールしてください。

## API・設定

### Q8: APIキーはどこで取得できますか？

**A**: [09_API_Acquisition_Guide.md](09_API_Acquisition_Guide.md) に詳しい手順があります。
最優先で準備すべきは：
1. Google Gemini API Key
2. AIVIS Speech（ローカルサーバーとしてダウンロード・起動）

### Q9: `.env` ファイルはどこに置けばいいですか？

**A**: プロジェクトのルートディレクトリ（`yukkuri_movie_maker_remotion` フォルダ直下）に配置してください。
Web GUI / CLI / API / Worker で共通利用します。

### Q10: APIキーが正しく設定されているか確認する方法は？

**A**: 
```bash
pnpm cli
```
を実行してください。エラーが出なければ正しく設定されています。

## 実行・動作

### Q11: 動画生成にどれくらい時間がかかりますか？

**A**: 5分の動画で約10〜30分程度です（PCスペックとAPI応答速度に依存）。
- テーマ選定・台本生成: 2〜5分
- 音声生成: 3〜10分
- 動画合成: 5〜15分

### Q12: 途中でエラーが出て止まりました。最初からやり直しですか？

**A**: いいえ。システムは各ステップごとに進捗を保存しています。
同じコマンドを再実行すると、完了したステップはスキップして続きから再開します。

### Q13: 生成された動画はどこに保存されますか？

**A**: 
- **最終動画**: `projects/[プロジェクトID]/files/video/output_video.mp4`
- **中間ファイル**: `projects/[プロジェクトID]/` 内の各フォルダ

### Q14: 複数の動画を同時に生成できますか？

**A**: 現在のバージョンでは1つずつの生成を推奨します。
並列実行は将来のバージョンで対応予定です。

## カスタマイズ

### Q15: キャラクターを変更できますか？

**A**: はい。`config/character_config.yaml` を編集し、`assets/characters/` に立ち絵素材を配置してください。
詳細は [10_Customization_Guide.md](10_Customization_Guide.md) を参照してください。

### Q16: 動画の長さを変更できますか？

**A**: はい。テーマ選定時に目標尺を指定できます。
設定ファイルまたはプロンプトで変更可能です。

### Q17: BGMを自分の好きな曲に変更できますか？

**A**: はい。`assets/audio/bgm/` に音楽ファイルを配置し、`config/audio_config.yaml` で指定してください。
**著作権に注意してください。**

## トラブルシューティング

### Q18: "ModuleNotFoundError" が出ます

**A**: 依存関係がインストールされていません。
```bash
pnpm install
```
を実行してください。

### Q19: 動画に音声が入っていません

**A**: 
1. 音声ファイルが正しく生成されているか確認（`projects/[ID]/files/audio/`）
2. FFmpegが正しくインストールされているか確認
3. ログファイル（`logs/`）でエラーを確認

### Q20: 文字化けします（Windows）

**A**: PowerShellで以下を実行してから起動してください：
```powershell
$OutputEncoding = [System.Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

### Q21: メモリ不足エラーが出ます

**A**: 
- 他のアプリケーションを終了してください
- 動画の解像度を下げてください（`config/encoding_config.yaml`）
- 並列処理数を減らしてください（`.env` の `MAX_WORKERS`）

**A**: はい、可能です。Google GeminiとAIVIS Speechはどちらも無料で利用できます。
- **Google Gemini**: 無料枠内で利用
- **AIVIS Speech**: ローカルサーバーとして無料で利用

**A**: 使用頻度によりますが：

## その他

### Q24: YouTube投稿機能はいつ実装されますか？

**A**: 現在開発中です。手動投稿で品質を確認してからの実装を予定しています。

### Q25: 商用利用は可能ですか？

**A**: システム自体は可能ですが、以下を確認してください：
- 使用するAPIサービスの商用利用規約
- 二次創作の場合は元作品のガイドライン
- BGM・効果音の商用利用可否

### Q26: バグを見つけた場合はどうすればいいですか？

**A**: 
1. ログファイル（`logs/`）を確認
2. エラーメッセージをコピー
3. 再現手順をまとめる
4. プロジェクトのIssueとして報告（GitHubを使用している場合）

### Q27: 新機能のリクエストはできますか？

**A**: はい。プロジェクトのIssueやDiscussionで提案してください。
