# 01. システム概要

## システムの目的
このシステムは、**「企画からYouTube投稿まで」**を全自動で行う「ゆっくり動画（Yukkuri Movie）」生成ツールです。
最新のAI技術（LLM、画像生成、音声合成）を組み合わせることで、人間が手作業で行っていた動画制作プロセスを自動化します。

## 主な機能

このツールは以下の工程を自動的に実行します：

1. **企画・テーマ選定**: 
   - LLM（Gemini/GPT-4）がトレンドや指定されたジャンルに基づいて動画のテーマを決定します。
2. **台本（スクリプト）作成**: 
   - 霊夢と魔理沙のようなキャラクター同士の掛け合い台本を作成します。
3. **音声合成 (TTS)**: 
   - AIVIS Speech 等をローカルサーバーとして使用し、キャラクターごとの音声を生成します。
4. **立ち絵アニメーション**: 
   - 音声に合わせて口パク（リップシンク）や瞬きをする立ち絵動画を生成します。
5. **背景・素材生成**: 
   - 台本の内容に合わせた背景画像や挿絵をAI（DALL-E 3/Stable Diffusion/Gemini）で生成します。
6. **動画編集・合成**: 
   - 音声、立ち絵、背景、字幕、BGMを合成し、一本の動画ファイル（MP4）として出力します。
7. **YouTube投稿** (実装予定): 
   - 完成した動画をYouTubeにアップロードします。

## 技術スタック

このシステムを再現するために使用されている主要な技術です。

- **プログラミング言語**: Python 3.11+
- **データベース**: SQLite (プロジェクト管理、データ保存)
- **動画処理**: FFmpeg (映像と音声の合成、エンコード)
- **画像処理**: Pillow, OpenCV
- **音声処理**: PyDub
- **AI/API**:
  - **LLM**: Google Gemini API (推奨), OpenAI API, Anthropic API
  - **TTS**: AIVIS Speech （ローカルサーバー、推奨）, Azure Speech Services
  - **画像生成**: Google Gemini Image Generation (推奨), DALL-E 3, Stable Diffusion

## アーキテクチャの概念

システムは「ワークフローエンジン」を中心に動作します。動画制作の各工程を「ステップ」として定義し、前のステップが完了したら次のステップへ進むという流れで処理が進みます。これにより、途中でエラーが発生しても、そのステップから再開することが可能です。
